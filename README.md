# Flutter Interview Test Task

## 1. Dependencies Used & Why

- **flutter**: The core Flutter SDK used for building the cross-platform mobile application.
- **cupertino_icons: ^1.0.8**: Provides iOS-style icons for consistent UI elements across platforms.
- **google_fonts: ^6.1.0**: Used for custom typography, specifically the 'multish' font family for a modern aesthetic.
- **get: ^4.6.6**: Employed for state management (GetX) and dependency injection, enabling reactive UI updates and simple routing.
- **flutter_svg: ^2.0.10**: Handles rendering of SVG icons and assets to ensure high-quality graphics on all screen densities.

## 2. Project Structure

